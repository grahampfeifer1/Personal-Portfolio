Projects

Minimum Spanning Tree Optimization

This project finds a Minimum Spanning Tree of any weighted graph. Prim's algorithm, Kruskal's algorithm, and an optimized Kruskal's algoithm are all tested using graphs with 100, 200, 400, 800, and 1600 vertices. To run this project, click Download Project on the left, unzip the folder, and in a terminal window run the command ./mst. The project also contains an excel file with some data and graphs related to the program. Source code for the project is available upon request.

Huffman Encoder/Decoder

This project uses Huffman coding to encode and decode text files. To run this project, click Download Project on the left, and in a terminal window, run the command ./encoder to see the usage of the program. The program works best if you add it to a folder with two or more text files to use while running the program. Source code for the project is available upon request.
